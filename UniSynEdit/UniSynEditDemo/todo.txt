1. Розработка графического отображения структуры документа.
1.1. Отображение начала свертка открытого и закрытого
1.2. Отображение линии связи начала и конца свертка
  для отображения линии связи необходимо опрделить размер блока отрисовки с учетом растояния между строками.

1.3. Отображение точки завершения свертка

2. Распознавание вложенных свертков
2.1. Отображение начала свертка внутри другого свертка в открытом и закрытом состоянии
2.2. Отображение конца свертка в открытом состоянии

3. Переопределение всех методов связанных с отрисовкой, которые будут срабатывать только если включена данная опция.

4. Организовать автоматизированный поиск блоков начала и конца свертков по заднным кретериям поиска. 
  Отображать в отдельном массиве данные о том что за тип строки. Просто строка 0, строка начала свертка открытого - 1,
  строка начала нового отрытого свертка свертка, внутри свертка - 2, строка закрытого свертка -3,
  строка закрытого свертка внутри свертка - 4, строка завершения открытого свертка - 5, строка внутри свертка - 6. 
  Строка завершения ткрытого свертка внутри свертка - 7.

5. Переопределение метода отрисовки самой области отображения текста. Цель не показывать текст скрытых свертков.
6. При копировании копировать содержимое скрытого свертка.
7. Добавить нумерации строк отдельной областью отображения. Показывать только в том случае,
  когда у нас указан параметр отображения.
